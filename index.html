Claude Command Center v1

Personal Operating Command Center

Claude is the command interface. This dashboard enforces one source of truth, classifies incoming work, and routes actions to Todoist, Calendar, Notion, Obsidian, or Drive.

Core Stack

  • Claude (command + reasoning)
  • Google Workspace (Gmail, Calendar, Drive)
  • Zapier MCP (action bus) + Google Apps Script
  • Notion, Todoist, Obsidian, Canva/Excel/PowerPoint

Source of Truth Rules

  • Task → Todoist
  • Time block/event → Google Calendar
  • Structured ops record → Notion
  • Private synthesis → Obsidian
  • Deliverable/export → Drive
